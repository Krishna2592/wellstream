# Grafana Dashboards

Import JSON dashboards into Grafana to visualize Kafka, Redis, and Spark metrics.

1. Open Grafana (default `http://localhost:3000`, user `admin`, password `admin`).
2. Navigate to **Dashboards → New → Import**.
3. Paste the JSON content of any dashboard you add here, or upload the file directly.
4. Set the Prometheus data source created in the setup guide.
